# Rob Laurie - Warm Boutique Design

**Design Direction:** Joanna Gaines aesthetic - warm, approachable, trustworthy expert

**Key Features:**
- Warm taupe + sage green + terracotta color scheme
- Story-driven, relationship-focused messaging
- Emphasizes local trust and transparency
- Warm, welcoming design
- Best for local trust-building

**Color Palette:**
- Primary: `#8b6f47` (Warm Taupe)
- Accent: `#d4a574` (Warm Sand)
- Green: `#7a9b6f` (Sage Green)
- Light: `#f8f6f1` (Cream)

**Deployed:** Vercel (auto-deploy from GitHub)

**View Live:** https://rob-design-variant-2.vercel.app

## Setup

1. Clone this repo
2. Open `index.html` in browser (or deploy to Vercel)
3. Update image URLs (see notes below)

## Image Placeholders

Currently using placeholder colors. To add real images:

1. Replace image URLs in `index.html`:
   - Hero image (right side): ~1200x900px
   - Story section image: ~700x600px
   - Before image (left side): ~600x600px
   - After image (right side): ~600x600px

2. Use Imgur, Cloudinary, or GitHub to host images

3. Update `src` attributes in HTML
